                                              Sqnqora - Real Time Chat Application

Synqora is a full-stack real-time chat application designed to deliver fast, secure, and seamless communication. It uses modern web technologies to provide instant messaging, authentication, and a smooth user experience. The project demonstrates real-world implementation of scalable architecture, real-time communication, and secure backend systems.

✨ Features

🔐 Custom JWT Authentication (No third-party auth)

⚡ Real-time messaging using Socket.io

🟢 Online/Offline user presence indicators

🔔 Notification & typing sounds (toggle option)

📨 Welcome emails on user signup

🖼️ Image upload support (Cloudinary

🧱 REST API with Node.js & Express

🍃 MongoDB for database management

🚦 API rate limiting for security

🎨 Responsive UI with React, Tailwind CSS & DaisyUI

🧠 Zustand for state management

🔧 Git & GitHub workflow (branches, PRs, merges)


🛠️ Tech Stack

↗️Frontend:

-React.js

-Tailwind CSS

-DaisyUI

-Zustand

↗️Backend:

-Node.js

-Express.js

-Socket.io

↗️Database:

-MongoDB

⚙️Other Tools:

-Cloudinary (image uploads)

-JWT (authentication)

🎯 Objective

The goal of Synqora is to build a secure, scalable, and efficient chat system while applying full-stack development concepts like authentication, real-time communication, and API design.
